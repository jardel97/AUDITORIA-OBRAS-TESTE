# Auditoria de Obras

## Visão Geral
Aplicação web para auditoria de serviços em obras, com registro de verificações de campo, validação de lançamentos no Prevision e CERP, e anexo de evidências fotográficas.

## Funcionalidades
- Cadastro de auditorias
- Registro de obra, bloco, unidade e serviço
- Identificação do executor e auditor
- Data da auditoria
- Status: Conforme / Não conforme / Pendente de ajuste
- Checklist de lançamento no Prevision, CERP e verificação in-loco
- Upload de fotos
- Dashboard com indicadores
- Consulta com filtros
- Exportação e importação em JSON

## Publicação
Este projeto pode ser publicado no GitHub Pages usando o arquivo `index.html`.
